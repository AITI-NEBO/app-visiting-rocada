# 📍 RocadaMed Telegram Bot

---

### 🚀 Описание

RocadaMed – это Telegram-бот, интегрированный с Битрикс24 (коробочная версия), который упрощает запись геолокации сотрудников в сделки типа "посещение". 

Благодаря боту ваша команда сможет оперативно фиксировать посещения прямо из чата!

---

## 🛠️ Установка и Настройка

После установки модуля перейдите в настройки вашего портала Битрикс24.

### ⚙️ Параметры Настройки

1. **🔑 Токен Телеграм-бота**  

   Укажите токен, чтобы система автоматически связала бота с вашим порталом.
  
2. **✉️ Сообщение приглашения**  

   Настройте текст приглашения для сотрудников.  

   Ссылка для приглашения будет сформирована автоматически.
  
3. **🏷️ Стадия сделки**  

   Укажите стадию, по которой бот будет фильтровать нужные сделки.
  
4. **🌍 Поля координат**  

   Определите, в какие поля сделки (широта и долгота) будут записываться данные геолокации.

👉 **Не забудьте сохранить настройки после заполнения всех параметров!**

## 🔔 Уведомления

Чтобы пригласить сотрудников в бот, используйте специальное поле с поиском по имени.  

Также можно открыть модальное окно структуры компании и выбрать нужных пользователей вручную.

- **📩 Системные уведомления**  

  Будут отправлены сотрудникам в их Битрикс24.
  
- **Важно**:  

  При повторном приглашении старая ссылка станет неактивной.
  
- **Авторизация**:  

  После перехода по ссылке сотрудник автоматически авторизуется в боте.

## 🤖 Работа в Telegram Боте

В боте будут отображаться сделки из выбранной в настройках стадии, доступные пользователю.  

Чтобы передать геолокацию:

1. **Выберите нужную сделку.**

2. **Нажмите кнопку "Передать гео"**.

📡 Бот автоматически отправит данные о местоположении в CRM.  

Координаты обновятся в сделке, и пользователь получит сообщение об успешном обновлении данных.

✨ Все просто и эффективно!

## 🗂️ Создаваемые Поля и Таблицы

При установке модуля создаются следующие элементы в системе:

- **HighloadBlock UserStageInfo**  

  Хранит состояние (временные данные) пользователей бота.
  
- **Поле UF_ROCADOMED_TELEGRAM_ID в сущности User**  

  Сохраняет Telegram ID пользователя.  

  Если поле заполнено, пользователь считается авторизованным в системе.
