# CRM RocadaMed — Мобильное приложение для торговых представителей

> PWA-приложение для управления визитами, клиентами и продажами стоматологической продукции.

**Демо:** [https://crm-rocadamed-pwa-271686259253.us-west1.run.app](https://crm-rocadamed-pwa-271686259253.us-west1.run.app)

---

## Назначение

Приложение предназначено для выездных торговых представителей компании RocadaMed. Оно помогает планировать рабочий день, управлять визитами к стоматологическим клиникам, фиксировать результаты встреч и отслеживать выполнение плана продаж.

## Ключевые возможности

### 📊 Дашборд
- Приветственный экран с прогрессом за день
- Статистика: завершённые визиты, количество заказов, выручка
- Быстрые действия: переход к визитам, карте клиентов
- Карточка ближайшего визита

### 📋 Управление визитами
- Список визитов на сегодня и завтра с фильтрацией по вкладкам
- Цветовая кодировка статусов: запланирован, в процессе, завершён
- Типы визитов: сбор заказа, презентация, консультация, новый клиент
- Детальная карточка визита с полной информацией

### 👤 ЛПР (лицо, принимающее решение)
- Контактные данные ЛПР на каждом визите: ФИО, должность
- Быстрые действия: звонок, email, Telegram
- Инфоповод визита — подсказка, о чём говорить с клиентом

### 🛒 Товары и заказы
- Список товаров с ценами в карточке визита
- Пометка «примерная сумма» для предварительных оценок
- Итоговая сумма заказа

### ✅ Завершение визита (3 шага)
1. **Геолокация** — подтверждение местоположения через GPS
2. **Результат** — выбор итога (заказ, отложен, отказ и др.), текстовый комментарий, голосовая заметка, фотографии
3. **Готово** — подтверждение отправки данных

### 🗺 Карта (Яндекс.Карты)
- Полноэкранная интерактивная карта
- Две вкладки: визиты на сегодня / все клиенты
- Цветные маркеры с привязкой к статусу
- Планирование визита к клиенту прямо с карты
- Кнопка «Моё местоположение»
- Выдвижная панель со списком точек

### 👨‍💼 Профиль и статистика
- Данные сотрудника: ФИО, должность, регион, контакты
- Статистика по периодам: сегодня / неделя / месяц
- Прогресс-бары выполнения плана
- Количество заказов и выручка по каждому периоду

### 🎨 Интерфейс
- **Светлая и тёмная тема** с переключателем в профиле
- Светлая тема установлена по умолчанию
- Адаптивный дизайн: мобильная версия с нижней навигацией + десктоп с боковой панелью
- PWA: установка на телефон, работа офлайн
- Анимации переходов между страницами

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Фронтенд | Vue 3 + Vue Router |
| Сборка | Vite + vite-plugin-pwa |
| Карты | Яндекс.Карты JS API 2.1 |
| Сервер | Express 4 (Node.js) |
| Хостинг | Google Cloud Run |
| Иконки | Google Material Symbols |
| Шрифт | Inter (Google Fonts) |

## Структура приложения

```
Главная (/)           — дашборд с прогрессом дня
Визиты (/visits)      — список визитов сегодня/завтра
Детали (/visits/:id)  — карточка визита, ЛПР, товары
Итог (/visits/:id/result) — завершение визита
Карта (/map)          — Яндекс.Карты с визитами/клиентами
Профиль (/profile)    — статистика, настройки, тема
```

## Текущий статус

Реализована демо-версия с моковыми данными. Приложение полностью работает как прототип для демонстрации функционала и UX. Данные не сохраняются на сервере — все действия происходят локально в браузере.

### Для полноценной работы потребуется:
- Интеграция с серверным API (бэкенд)
- Авторизация через CRM / Telegram
- Синхронизация данных о визитах и заказах
- Хранение голосовых заметок и фотографий
- Push-уведомления о предстоящих визитах
