# 🚀 RocadaMed PWA — Мобильное приложение для въездных сотрудников

---

### 📱 Описание

РМ-Приложение — это PWA для выездных торговых представителей (sales-менеджеров) и сервисных инженеров компании RocadaMed.
Приложение интегрировано с Битрикс24 и предназначено для оперативного управления визитами к клиентам (клиникам), просмотра карточек контактов и компаний, а также фиксации геолокации и комментариев.

**Особенности MVP:**
- Просмотр списка визитов (сделок) на "Сегодня" и "Завтра" (по заданным стадиям).
- Интеграция с картами для визуализации расписания и маршрута.
- Авторизация сотрудников через стандартное окно входа Битрикс24 с наследованием прав доступа (D7 API).
- Загрузка фотографий и файлов прямо с телефона в CRM.

### 🛠️ Технологический стек

- **Фронтенд:** Vue 3 + Vue Router
- **Сборка:** Vite + vite-plugin-pwa
- **Карты:** Яндекс.Карты JS API 2.1
- **Хостинг:** Timeweb Cloud

### 💻 Как запустить локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/AITI-NEBO/app-visiting-rocada.git
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите dev-сервер:
   ```bash
   npm run dev
   ```
